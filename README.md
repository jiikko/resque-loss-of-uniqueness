# README

This is the reproduction code for https://github.com/resque/resque-unique_at_runtime/issues/194

## Setup

- bundle install

## How to Run

- bundle exec foreman s
- bundle exec rails c
  - `5000.times { TheWorker.perform_async }`
- `tail -F log/development.log | grep ERROR`
